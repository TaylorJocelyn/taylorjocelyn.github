C++ STL(笔记篇)

2021-07-07 PV:

 最近学了STL 发现C++真香~ 第一篇博客就来记个笔记叭！

  • vector

    向量（变长数组） 利用倍增思想实现 Hint:内存分配时间与申请次数有关而与内存大小关系不大

    • size() 返回元素个数 时间复杂度O(1)

      vector<int> vi; cout<<vi.size() ;

    • empty() 返回是否为空

      vi.empty()

    • clear() 清空 时间复杂度O(N)

      vi.clear()

    • front()/back() 取第一个元素/取最后一个元素

    • push_back()/pop_back() 在尾部插入一个元素/弹出尾元素

    • begin()/end()
    • 支持随即寻址 [ ]
    • 支持比较运算，按字典序

  • pair

    • 初始化

    • 支持比较运算，以first为第一关键字，以second为第二关键字（字典序）

  • string

    • size()/length() 返回字符串长度

      string s; int len=s.leng();

    • empty()

    • clear()

    • substr(起始下标，(子串长度)) 返回子串

    • c_str() 返回字符串所在字符数组的起始地址

    • erase s.erase(first,last); 左闭右开区间 [ )

  • queue

    • size()

    • empty()

    • push() 向队尾插入一个元素

    • front() 返回队头元素

    • back() 返回队尾元素

    • pop() 弹出队头元素

  • priority_queue

    默认是大根堆

    • push() 插入一个元素

    • top() 返回堆顶元素

    • pop() 弹出堆顶元素

    • 优先级设置

      • 数字越大优先级越大

      • 数字越小优先级越大（小根堆）

      • 结构体优先级设置

  • stack

    • size()
    • empty()
    • push() 向栈顶插入一个元素
    • top() 返回栈顶元素
    • pop() 弹出栈顶元素

  • deque

    双端队列

    • size()
    • empty()
    • clear()
    • front()/back()
    • push_back()/pop_back()
    • push_front()/pop_front()
    • begin()/end()
    • 支持随机寻址 [ ]

set, map, multiset, multimap, 基于平衡二叉树（红黑树），动态维护有序序列

  • set/multiset

    • insert() 插入一个数 并自动排序和去重

    • find() 查找一个数 返回一个迭代器 时间复杂度O(logN)

      cout<<*(st.find(1));

    • count() 返回某一个数的个数

    • erase()

      (1) 输入是一个数x，删除所有x   O(k + logn)
(2) 输入一个迭代器，删除这个迭代器

    • lower_bound()/upper_bound()

      lower_bound(x)  返回大于等于x的最小的数的迭代器
upper_bound(x)  返回大于x的最小的数的迭代器

  • map/multimap

    • insert() 插入的数是一个pair

    • erase() 输入的参数是pair或者迭代器

    • find()

    • 注意multimap不支持此操作 时间复杂度是 O(logn)

    • lower_bound()/upper_bound()

  • **unordered_set, unordered_map, unordered_multiset, unordered_multimap, 哈希表 **

    1.和上面类似，增删改查的时间复杂度是 O(1)

    2.不支持 lower_bound()/upper_bound()， 迭代器的++，–

  • bitset

    压位 占用空间为原来的1/8

    • 初始化

      bitset<10000> s;

    • 支持 ~, &, |, ^，>, <<，==, !=，[ ] 等操作

    • count() 返回有多少个1

    • any() 判断是否至少有一个1

    • none() 判断是否全为0

    • set() 把所有位置成1

    • set(k, v) 将第k位变成v

    • reset() 把所有位变成0

    • flip(k) 等价于~k

中间记的略简洁 就先这样叭！
